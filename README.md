# ShadowSuite
A suite for installing hacking tools and building payloads, malwares, etc. (Use ethically)

# HOW TO INSTALL
- Scroll to bottom or right side of this page
- Click on <b>Releases</b>
- Select latest version
- Download ShadowSuite.zip
- Extract the downloaded ZIP to a folder, e.g Desktop
- Start "run.exe" to start the application
